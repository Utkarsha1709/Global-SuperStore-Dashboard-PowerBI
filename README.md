## Project Overview
This project uses the Global Superstore dataset, which provides comprehensive information about global sales, profits, orders, and customer demographics across different countries and regions. Using Power BI Desktop, the goal is to perform an in-depth sales performance analysis, identify key profit drivers, and provide actionable insights for business decision-making.
Dataset Description
The dataset includes data such as:
•	Order ID, Order Date, Ship Date
•	Customer Name, Segment, and Region
•	Country, City, and State
•	Product Category and Sub-Category
•	Sales, Quantity, Discount, and Profit

## Business Objectives
The primary objectives of this analysis are:
•	Analyze global sales and profit distribution by region and country.
•	Identify top-performing categories and sub-categories contributing most to revenue.
•	Examine customer segment performance across markets.
•	Study discount impact on profit margins.
•	Monthly trends in sales growth.
•	Build an interactive Power BI dashboard with filters and slicers for deeper exploration.

## Tools & Techniques
Microsoft Power BI
•	Data Cleaning & Transformation: Performed using Power Query to handle missing data, remove duplicates, and standardize fields like country and date.
•	DAX Calculations: Created measures for Total Sales, Total Profit, Total Quantity , Total Customers, Total Orders.
•	Visualizations:
•	Region wise Sales [Map Chart]
•	Category Wise Sales & Profit [Bar Chart]
•	Segment Wise Sales Distribution [Pie Chart]
•	Monthly Sales Trend [Line Chart]
•	KPI Card: Total Orders, Sales, Customers, Quantity Sold, Profit.

## Deliverables
•	Power BI dataset with applied transformations and calculated fields.
•	Fully interactive Power BI Dashboard.
•	KPI Cards summarizing Total Sales, Profit, and Quantity Sold, Customers, Order Sales
•	Region-wise and Category-wise performance reports.
